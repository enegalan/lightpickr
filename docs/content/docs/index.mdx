**Lightpickr** is a small, dependency-free date and range picker for the browser: calendar grid, month/year drill-down, optional time panel, CSS-variable theming, render hooks, and a **plugin** API.

Use a text **input** or **textarea** for a popover, or any other element (e.g. a `div`) for an **inline** calendar when `inline` is omitted or set to `true`.

## Install

```npm
npm install lightpickr
```

## Quick start

```js
import Lightpickr from 'lightpickr';
import 'lightpickr/lightpickr.css';

new Lightpickr('#my-input');
```

The field value updates when the user changes the selection. Use **`range: true`**, **`enableTime: true`**, and other options from the [Options](/docs/documentation/options) reference.
