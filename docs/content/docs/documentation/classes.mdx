The shipped stylesheet uses the **`lp`** prefix.

<heading>
## options.classes
<definition><param-type><purple>Partial</purple>&lt;LightpickrClassMap&gt;</param-type></definition>
</heading>

Each value is a **class name string**.

<heading>
## options.attributes
<definition><param-type><purple>Record</purple>&lt;string, string&gt;</param-type></definition>
</heading>

Attribute **names**. Merge with `options.attributes` if you need to avoid collisions.

| Key            | attribute name          | Used on                                   |
|----------------|-------------------------|-------------------------------------------|
| `root`         | `data-lp-root`          | Root `.lp`                                |
| `nav`          | `data-lp-nav`           | `prev` / `next` / `title` on nav controls |
| `pointerMain`  | `data-lp-pointer-main`  | Pointer placement (top/bottom/left/right) |
| `pointerSec`   | `data-lp-pointer-sec`   | Pointer secondary axis                    |
| `day`          | `data-lp-day`           | Day cells                                 |
| `dayName`      | `data-lp-day-name`      | Weekday header buttons when clickable     |
| `time`         | `data-lp-time`          | Time inputs / clock                       |
| `month`        | `data-lp-month`         | Month cells                               |
| `year`         | `data-lp-year`          | Year cells                                |
| `footerAction` | `data-lp-footer-action` | Preset footer buttons                     |
| `footerBound`  | `data-lp-footer-bound`  | Footer binding marker                     |

<heading>
## options.properties
<definition><param-type><purple>Record</purple>&lt;string, string&gt;</param-type></definition>
</heading>

Maps **logical keys** to **CSS custom property names** (including the `--` prefix).

| Key                     | CSS variable                   | Notes                                      |
|-------------------------|--------------------------------|--------------------------------------------|
| `pointerVisible`        | `--lp-pointer-visible`         | `0` / `1` toggles arrow                    |
| `pointerSize`           | `--lp-pointer-size`            | Arrow box size                             |
| `popoverGapX`           | `--lp-popover-gap-x`           | Horizontal gap for default placement       |
| `popoverGapY`           | `--lp-popover-gap-y`           | Vertical gap                               |
| `popoverViewportMargin` | `--lp-popover-viewport-margin` | Min inset from viewport edge when clamping |
