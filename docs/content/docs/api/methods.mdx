<heading>
### selectDate
<definition><param-type><gray>(</gray>date, opts?<gray>)</gray> <brown>=></brown> <blue>void</blue></param-type></definition>
</heading>

**Single / multiple:** one value or array; each day runs **<code><purple>onBeforeSelect</purple></code>**.

**Range:** **<code><gray>[[</gray>start<gray>,</gray> end<gray>],</gray> ...<gray>]</gray></code>** replaces all ranges.

**`opts`:** <code>&#123; close?<brown>:</brown> <red>boolean</red> &#125;</code> — after a programmatic range set, <code>close<brown>:</brown> <red>true</red></code> may trigger **<code><purple>hide</purple><gray>()</gray></code>** with **<code><purple>closeOnSelect</purple></code>**.

### formatDate
<definition><param-type><gray>(</gray>date, format?<gray>)</gray> <brown>=></brown> <green>string</green></param-type></definition>
</heading>

Formats date in passed format.

Uses **<code><purple>format</purple></code>** when it is a **<code><green>string</green></code>** (full token set in [Options — format](/docs/documentation/options#format)). If that option is a **<code><purple>function</purple></code>**, it is invoked with a single **<code><blue>Date</blue></code>** derived from **<code><purple>date</purple></code>** and current time state.
